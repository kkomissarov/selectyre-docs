    📦 Структура JSON-файла с товарами

    JSON-файл с ассортиментом товаров содержит информацию о шинах и дисках в формате JSON.

    📖 Подробное описание всех полей см. в разделе Структура данных

    Общая структура

    {
  "metainfo": { ... },
  "tires": [ ... ],
  "wheels": [ ... ]
}

    🧠 metainfo — Метаданные

    Раздел с общей информацией о выгрузке. Описание полей см. в Структуре данных → Метаданные.

    📌 Пример:

    "metainfo": {
  "utc_update_time": "2025-04-01T19:20:20.861Z",
  "client_name": "mrnr",
  "license_expire": "2025-04-01"
}
